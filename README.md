## Key Findings

- Paradox of Choice (2/10): 19 training tools listed with no grouping
- Occams Razor (2/10): Three sub-sites for one product
- Blank white voids: 500-2000px empty space between sections
- Educator page: one off-site CTA, no login, no lead capture
- Registration form renders below full-viewport white void
- Access code redemption: 5+ steps, tutorial video needed
